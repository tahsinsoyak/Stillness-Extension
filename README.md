# Stillness Extension ⏳

> **A mindful pause before digital decisions.**

Stillness is a premium, local-only Chrome extension that helps you slow down before making impulsive online decisions. Featuring a cinematic "Ocean Breath" atmosphere, it creates a gentle space between impulse and action.

## Core Features ✨
- **Ocean Breath Atmosphere:** Multiple organic, animated liquid waves with deep parallax movement.
- **Naïf Zen Design:** Soft glassmorphic UI elements and refined typography for a gentle experience.
- **Custom Rules:** Set rules for specific domains (e.g., amazon.com, x.com) based on action types.
- **Instant Cooldown:** Trigger a manual pause on any website directly from the popup.
- **Privacy First:** 100% local storage. No accounts, no tracking, and zero data leaves your browser.

## Tech Stack 🛠
- Chrome Extension (Manifest V3)
- React 18 & TypeScript
- Vite & Tailwind CSS
- Framer Motion (Cinematic Animations)
- Zustand (State Management)

## Getting Started 🚀

### Build Instructions
1. Install dependencies: `npm install`
2. Build the project: `npm run build`
3. Load the `dist/` folder as an unpacked extension in Chrome (`chrome://extensions`).
